# Sistema de Cadastro em Python

Meu primeiro projeto de portfólio desenvolvido durante os estudos do curso de Python Mundo 3 do Curso em Vídeo, utilizando conceitos de modularização, tratamento de exceções e manipulação de arquivos.

## Funcionalidades

- Cadastrar pessoas
- Listar pessoas cadastradas
- Armazenamento em arquivo `.txt`
- Tratamento de erros
- Menu interativo

## Tecnologias utilizadas

- Python 3
- PyCharm
- Git
- GitHub Desktop

## Estrutura do projeto

```
SistemaCadastroPython
│
├── lib
│   ├── arquivo
│   └── interface
│
├── sistema.py
└── projeto.txt
```

## Objetivo

Este projeto representa meu primeiro projeto de portfólio. Pretendo evoluí-lo ao longo dos estudos adicionando novas funcionalidades, como:

- Pesquisa de pessoas
- Exclusão de registros
- Edição de cadastros
- Migração para SQLite
- Interface gráfica
- API REST
